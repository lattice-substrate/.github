# Lattice Substrate

Infrastructure primitives for deterministic data processing.

**[json-canon](https://github.com/lattice-substrate/json-canon)** -- RFC 8785 JSON
Canonicalization Scheme for Go. Strict RFC 8259 parser, canonical serializer,
stable CLI ABI. Zero external dependencies.

[Engineering articles](https://lattice-substrate.github.io) on standards
conformance, deterministic systems, and the decisions behind infrastructure
software.
